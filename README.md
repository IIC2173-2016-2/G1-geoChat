Tarea IIC2173
=============

Distributed chat created with [deepstream.io](http://deepstream.io)

#### Instalar dependencias
`$ npm install`

#### Iniciar cliente
`$ npm run start`

#### Iniciar servidor
`$ npm run server`

# Technologies

  - [deepstream.io](http://deepstream.io)
  - [React](https://facebook.github.io/react/)
